## English

This directory contains summary documents of major changes made to the project during development.

### Purpose

- Document significant project milestones
- Provide context for major refactorings or updates
- Keep a detailed history of important decisions
- Help new contributors understand the project evolution

### Contents

- **PROJECT_SUMMARY.md** - Initial project setup summary
- **BILINGUAL_UPDATE.md** - Documentation bilingual conversion summary

### Note

These files are for reference and documentation purposes. They complement the CHANGELOG.md but provide more detailed context about the changes.
